# Swiggy-Sales-Market-Intelligence-Analysis
Analyzed 150K+ food orders (₹5Cr+ GMV) using Python EDA to uncover pan-India trends across sales, geography, and cuisine. Built KPIs and visualizations to evaluate regional performance, identifying top revenue cities and underserved markets for data-driven decisions.
Used Python Libraries: pandas,numpy,matplotlib,seaborn,plotly to create visualization charts
